# react-step-by-step

## step05 react-router

> **关键点**
>
> - `react-router` - [教程](https://github.com/zp1024/react-router-notes)

**运行方法**

- **运行开发环境**

```sh
$ npm install
$ NODE_ENV=dev # 如果是windows环境，应执行：set NODE_ENV=dev
$ npm start
```

- **运行生产环境**

```sh
$ npm install
$ NODE_ENV=prod # 如果是windows环境，应执行：set NODE_ENV=prod
$ npm start
```
